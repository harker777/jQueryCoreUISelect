jQuery сoreUISelect

jQuery сoreUISelect is a cross browser easy to stylize select element with jQuery and CSS

GitHub Page
Download ZIP

Available Features:

  • Full customization
  • Optiongroup support
  • Automatic calculations
  • Keyboard support
  • Powerful callback functions
  • Compatible with mobile devices
  • Support jScrollPane plugin for customize default scrollbar

Default settings

        $('#countries').сoreUISelect();

With jScrollPane plugin

        $('#countries_2').сoreUISelect({
            jScrollPane : {
                verticalDragMinHeight: 20,
                verticalDragMaxHeight: 20,
                showArrows : true
            }
        });

Append to body setting

Custom dropdown build in body

        $('#countries_3').сoreUISelect({
            appendToBody : true
        });

Callback functions and API

            onInit : addсoreUISelectListener,
            onOpen : addсoreUISelectListener,
            onClose : addсoreUISelectListener,
            onChange : addсoreUISelectListener,
            onDestroy : addсoreUISelectListener
        });
